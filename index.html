💡 Tips for Better Diagnoses

  • Include the full log — the first error line matters most, not the loudest.
  • Select the correct CI platform from the dropdown before running.
  • Include any relevant config files (package.json, .github/workflows/*.yml, Dockerfile).
  • If you see "exit code 1" with many errors, the real cause is almost always the first error.
